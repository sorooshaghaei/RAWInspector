## Description

RAW Inspector is a simple local utility for photographers, used-camera buyers, sellers, and small camera shops.

Choose original RAW files from the Files app and inspect useful camera metadata directly on your iPhone or iPad. The app can read Fujifilm RAF Image Count from MakerNote data, show camera model and firmware/software information, verify the file type, warn when metadata is missing, and export a PDF or PNG report for used-camera sales.

Main features:

- Fujifilm RAF Image Count / shutter count reading
- Batch inspection for multiple RAW files
- Camera model and brand
- Firmware/software field
- Lens used when available
- Serial number when available
- Capture date when available
- File type verification
- Missing metadata and edited/exported file warnings
- Seller/buyer PDF report export
- Seller/buyer PNG report export
- Local-only processing

Privacy: RAW files are processed on your device. The app does not upload images, does not use analytics, does not show ads, and does not collect personal data.

Important: shutter count is not standardized across all camera brands. Fujifilm RAF support is the primary shutter-count function. Nikon support is best-effort where MakerNote data exposes a readable value. Canon and Sony files often do not expose a reliable shutter count in ordinary RAW metadata, so the app may show metadata without a shutter count.

This app is an independent utility and is not affiliated with, endorsed by, or sponsored by Fujifilm, Canon, Nikon, Sony, or any camera manufacturer.
